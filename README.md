# DataFest Healthcare Analytics Project

## Overview
This repository is a case study of a healthcare data analysis project completed during the latest DataFest competition.

As part of a team, I worked on analyzing a large real-world hospital dataset provided in raw form. Because the data was confidential, it is not included in this repository. Instead, this case study highlights the project context, my role, the tools we used, and the type of analytical work involved.

## Project Context
The competition challenged participants to work with raw hospital data and transform it into meaningful findings. The goal was to explore the data, identify important patterns, and generate useful insights that could support better understanding and decision-making.

## My Role
I contributed to the analysis process as part of the team, working with Python and RStudio to:
- explore and interpret raw data
- identify meaningful patterns
- help turn complex information into useful insights
- support data-driven thinking around real-world healthcare challenges

## Tools Used
- Python
- RStudio
- Data cleaning
- Exploratory data analysis
- Team-based analytical workflows

## Key Takeaways
- Real-world data often requires careful exploration before insights become visible
- Strong analysis is not only about coding, but also about asking useful questions
- Team collaboration played an important role in shaping the final insights
- Working with healthcare data strengthened my interest in data science and practical analytics

## Note on Data Privacy
The original dataset is not shared in this repository because it was provided under competition restrictions and contains protected information.
